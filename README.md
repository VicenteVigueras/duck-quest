# Duck Quest

A 2D action game prototype built in C with [Raylib](https://www.raylib.com/).

## Requirements

- macOS
- GCC
- Raylib (`brew install raylib`)

## Build & Run

```bash
make run
```

Other targets: `make`, `make clean`, `make rebuild`.

## Controls

| Key | Action |
|-----|--------|
| WASD | Move |
| SPACE | Attack (when holding sword) |
| R | Restart |
